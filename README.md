### Student-Course Management MicroService-1

Student-Course Management MicroService-1 is a Spring Boot–based microservice designed to manage academic resources such as students, instructors, and courses. The service exposes RESTful APIs for performing CRUD operations and implements a layered architecture including controller, service, repository, DTO, and entity layers.

The application uses Spring Security for authentication and authorization and demonstrates best practices such as global exception handling, DTO-based data transfer, and external service communication.

Key Features:

* REST APIs for Student, Instructor, and Course management
* Secure authentication using Spring Security
* Layered architecture (Controller → Service → Repository)
* DTO pattern for API data transfer
* Global exception handling
* JPA/Hibernate for database interaction
* External microservice communication using REST clients

Tech Stack:

* Java
* Spring Boot
* Spring Security
* Spring Data JPA
* Maven
* RESTful APIs
